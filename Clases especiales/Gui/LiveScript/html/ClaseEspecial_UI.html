Introduccin

La interfaz grfica de usuario, conocida tambin como GUI (del ingls graphical user interface), es un programa que acta de interfaz de usuario, utilizando un conjunto de imgenes y objetos grficos, llamados componentes, para representar la informacin y acciones disponibles en la interfaz. Su principal uso, consiste en proporcionar un entorno visual sencillo para permitir la comunicacin con el sistema.

Los componentes de una interface de usuario pueden estar comprendidos en mens, barras de herramientas, botones, listas y controles deslizantes, por nombrar algunos. Las creadas por Matlab pueden llevar a cabo cualquier tipo de procesamiento de cmputos, leer y escribir archivos, comunicarse con otras interfaces y presentar tablas y/o grficos.

La misma contiene:

  • Un componente de ejes.
  • Un men pop-up.
  • Texto esttico.
  • Botones.

MATLAB permite desarrollar front-ends GUI que automatizan una tarea o clculo. La GUI normalmente contiene controles tales como mens, barras de herramientas, botones y controles deslizantes. Muchos productos de MATLAB, como Curve Fitting Toolbox , Signal Processing Toolbox y Control System Toolbox incluyen aplicaciones con interfaces de usuario personalizadas.

MATLAB permite tres caminos para la programacin de GUI:

  1. GUIDE (Asistente).
  2. Programando (UIControls).
  3. App Designer.

Cmo trabajan las GUI?

Las interfaces esperan que el usuario manipule un control y responden a su accin de a turnos. Los controles y la UI misma tienen uno o ms callbacks que se encargan de "llamar" a Matlab para llevar a cabo cierta tarea. Una accin particular del usuario, como podra ser el click de un botn, desencadanea la ejecucin de un callback. Por lo tanto, el creador de GUI's debe encargarse de escribir callbacks que definirn qu es lo que producen los diferentes componentes de una interface de usuario.

Programando (UIControls)

El siguiente ejemplo ilustra cmo programar una GUI muy sencilla en Matlab. Los tpicos que se ven a continuacin lo guiarn a travs del proceso de creacin de una interfaz de usuario.

Primero debe crearse una funcin y la misma debe ser guardada en la carpeta de trabajo correspondiente.

Crear figura

El primer paso luego de creado el archivo .m de funcin consta de crear una figura con las siguientes lneas de cdigo:

f = figure('Visible','off','Position',[360 200 450 285]);

A la figura se le establece una posicin correspondiente y se la declara invisible mientras es construida.

La propiedad 'Visible' establece que el usuario pueda o no ver a la figura. En este ejemplo, la figura se har visible para el usuario una vez que todos los componentes de la interfaz hayan sido inicializados.

La propiedad 'Position' un vector fila de cuatro elementos. Cada uno de ellos establece una posicin de localizacin de la figura en cuestin. El primer elemento hace referencia a la distancia desde la izquierda, el segundo a la distancia desde abajo, el tercero al ancho de la figura y el cuarto a su largo.

Agregar componentes (uicontrols)

  • Se crean los siguientes botones:

hsurf = uicontrol('Style', 'pushbutton',...

'String','Surf','Position',[315 220 70 25]);

hmesh = uicontrol('Style', 'pushbutton',...

'String','Mesh','Position',[315 180 70 25]);

hcontour = uicontrol('Style', 'pushbutton',...

'String','Contour','Position',[315 135 70 25]);

Cada una de las declaraciones utiliza una serie de pares propiedad-valor del objeto uicontrol para definir a cada uno de los botones.

  • Style establece el tipo de componente, en este caso un botn.
  • String especifica el texto que aparece en el botn.
  • Position establece la posicin del botn en la figura y sus dimensiones.

Cada uicontrol es utilizado con un handle del botn creado.

  • Se crea el men pop-up junto con su texto esttico:

htext = uicontrol('Style', 'text',...

'String','Select Data','Position',[325 90 60 15]);

hpopup = uicontrol('Style', 'popupmenu',...

'String',{'Peaks','Membrane','Sinc'},'Position',[300 50 100 25]);

Al componente de men pop-up en su propiedad String se le establece un valor de cell array de tres elementos, cada uno especificando los distintos items del men.

  • Agregar ejes para grfica.

ha = axes('Units','pixels','Position',[50, 60, 200, 185]);

Se establece que las unidades del objeto sean en pixels para que todos los componentes tengan las mismas.

Por ltimo, se declara visible a la figura:

Cada componente uicontrol tiene una propiedad callback asociada. Hay que especificar el nombre de la misma con el evento asociado. Se hace de la siguiente manera para este ejemplo:

Se ha aadido el valor de la propiedad callback del botn uicontrol cuyo handle es hsurf. Callback es el nombre de la propiedad y surfbutton_Callback es la funcin que se llama para ejectuar una accin cuando se utiliza ese componente.

Creacin de figuras

Las interfaces de usuario son figuras. Lo primero que se hace es crear una figura, antes de crear cualquier tipo de componente, y asignarle un handle a la misma.

fh = figure('Visible','on') % fh es el handle de la figura.

Las propiedades ms comunes de una figura son:

  • MenuBar (valores: 'figure'/'none')
  • Name (valores: String)
  • NumberTitle (valores: 'on'/'off')
  • Position (vector de 4 elementos [dist. izq., dist. abajo, ancho, largo])
  • Resize (valores: 'on'/'off')
  • Toolbar (valores: 'auto'/'none'/'figure')
  • Units ('pixels'/'centimeters'/'characters'/'inches'/'normalized'/'points')
  • Visible (valores: 'on'/'off')

fh.MenuBar = 'none';

fh.Name = 'Figura de prueba';

fh.NumberTitle = 'off';

fh.Resize = 'off';

Componentes disponibles

Los controles de una interface de usuario son componentes UI; pueden ser botones, check boxes y controles deslizantes, entre otros; tablas que presenten datos en filas y columnas; paneles que agrupen elementos relacionados.

Push button

El siguiente cdigo crea un botn:

f = figure;

pb = uicontrol(f,'Style','pushbutton','String','Botn 1',...

'Position',[50 20 60 40]);

El primer argumento de uicontrol especifica el partenesco; o sea, a qu estructura pertenece el botn. En este caso, es una figura pero podra ser cualquier otro contenedor como sea un panel o un grupo de botones. Con 'Style','pushbutton' se establece que el componente es un botn.

Radio button

El siguiente cdigo crea un radio button:

f = figure;

r = uicontrol(f, 'Style', 'radiobutton', ...

'String', 'Radio button 1', 'Position', [30 20 150 20]);

Toggle button

El siguiente cdigo crea un toggle button, el mismo es un botn que elige entre dos posibles modos o estados:

f = figure;

tb = uicontrol(f, 'Style', 'togglebutton',...

'String', 'L/R', 'Position', [30 20 100 30]);

Check Box

El siguiente cdigo crea un check box:

f = figure;

cb = uicontrol(f, 'Style', 'checkbox', ...

'String', 'Check box 1', ...

'Value',1,'Position', [30 20 130 20]);

La propiedad Value indica si el check box est o no tildado por defecto. Si el valor es 0, la caja aparece destilada mientras que si el valor es 1, ocurre lo contrario.

Slider

El siguiente cdigo crea un control deslizante:

f = figure;

s = uicontrol(f, 'Style', 'slider',...

'Min', 0, 'Max', 100, 'Value', 25, ...

'SliderStep', [0.05 0.2], ...

'Position', [30 20 150 30]);

Min y Max especifican el rango de valores entre los que toma el control deslizante. El primer valor debe ser ms chico que el primero. Value especifica el valor por defecto en el que est establecido el control deslizante una vez que se ejecuta la UI.

SliderStep es un array de dos elementos. El primero especifica la cantidad fraccional que se desliza el control cada vez que se haga click sobre alguna de las flechas laterales (en este caso, es del 5 por ciento). El segundo especifica la cantidad fraccional que se desliza el control cuando se hace click sobre el canal (en este caso, es del 20 por ciento).

Texto esttico

El siguiente cdigo crea un texto esttico:

f = figure;

t = uicontrol(f, 'Style', 'text', ...

'String', 'Texto esttico', ...

'Position', [30 50 150 30]);

Editable text field

El siguiente cdigo crea un campo de texto editable por el usuario:

f = figure;

txtbox = uicontrol(f, 'Style', 'edit',...

'String', 'Ingrese su mensaje', ...

'Position', [30 50 130 20]);

Para permitir que haya mltiples lneas de texto el valor de Max - Min tiene que ser mayor a 1. Si es menor o igual a 1, Matlab slo muestra parte del texto escrito cuando el mismo supera las dimensiones del campo donde se encuentra el texto.

Pop-up menu

El siguiente cdigo crea un men pop-up.

f = figure;

pm = uicontrol(f, 'Style', 'popupmenu',...

'String', {'uno', 'dos', 'tres', 'cuatro'},...

'Value', 2, 'Position', [30 80 130 20]);

En este caso, la propiedad String tiene como valor a un cell array cuyos elementos son strings que se corresponden con los diferentes tems del men. El valor de la propiedad Value indica cul de los elementos se muestra por defecto.

List Box

El siguiente cdigo crea una lista:

f = figure;

lb = uicontrol(f, 'Style', 'listbox',...

'String', {'uno', 'dos', 'tres', 'cuatro'},...

'Position', [30 20 180 80], 'Max', 2, 'Min', 0,'Value', [1 3]);

Para que el usuario pueda seleccionar ms de un elemento. Max - Min debe ser mayor que 1. Estableciendo un vector como valor de la propiedad Value, se seleccionan por defecto los elementos indicados. Si se quiere que no haya una seleccin inicial, el valor de la propiedad Value debe ser una matriz vaca [ ].

Tables

El siguiente cdigo crea una tabla a partir de los datos obtenidos a partir de la funcin magic(5).

f = figure;

tb = uitable(f, 'Data', magic(5));

Pueden ser ajustados el ancho y el largo de la tabla para acomodar la extensin de los datos contenidos en la misma. La propiedad Position controla los lmites externos de la tabla mientras que la propiedad Extent indica la extensin de los datos.

tb.Position(3) = tb.Extent(3);

tb.Position(4) = tb.Extent(4);

Pueden cambiarse otras alternativas:

  • Para que el usuario pueda editar las celdas se trabaja con la propiedad ColumnEditable.
  • Para especificar los nombres de las filas y de las columnas se trabaja con las propiedades RowName y ColumnName.
  • Para especificar el formato de los datos contenidos se trabaja con la propiedad ColumnFormat.

'Units', 'normalized', 'Position', [.1 .15 .8 .3]);
