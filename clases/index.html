    Formato¶

    Las clases usan Marimo — notebooks reactivos en archivos .py:

    # Abrir en modo edición
marimo edit clase_01/contenido.py

# Ejecutar en modo lectura
marimo run clase_01/contenido.py

    Cada clase contiene: - README.md — Objetivos y resumen - contenido.py — Notebook principal (Marimo) - ejercicios.py — Ejercicios (Marimo) - soluciones.py — Soluciones (liberado post-clase) - extra/ — Recursos adicionales
